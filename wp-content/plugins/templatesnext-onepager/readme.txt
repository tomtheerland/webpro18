Creates One-Page home/landing with highly customizable and sortable sections like about us, services, portfolio, team, contact us, etc.

== Description ==

Create onepage web sites or landing pages in few minutes. Custom Portfolio, Testimonial, Team and Contact Us Sections for TemplatesNext Wordpress Themes.  It creates easy to customize, edit sections on front page.

[Demo and documentation &raquo;](http://templatesnext.in/demo/)

Some of the features may not work properly with any other themes. 

[Best suits with WP theme i-amaze &raquo;](http://templatesnext.in/i-amaze-landing/) 
[And WordPress Theme i-ONE &raquo;](http://templatesnext.in/i-one-landing/)

= Theme direct download GitHub URLs : = 
i-one : https://github.com/TemplatesNext/i-one/archive/master.zip
i-amaze : https://github.com/TemplatesNext/i-amaze/archive/master.zip


== Installation ==

Install the plugin into the `/wp-content/plugins/` folder, and activate it.
